# Templater

My own template system so i wont have to manage templates manually

# Usage

- tmpl new <name> - takes the current dir and turns it into a template with name <name>
- tmpl use <name> - applies the template <name> on your root directory
- tmpl del <name> - deletes template <name> (IRREVERSIBLE)
- tmpl mv <oldname> <newname> - renames the template

# Notes

currently, this is linux only, may not work on windows
